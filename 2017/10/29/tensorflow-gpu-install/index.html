安装 Tensorflow

由于 Tensorflow 网站结构或安装方式总是变化，所以不再给出详细安装过程。

Tensorflow 官方网站

Tensorflow 官方安装教程

Windows 用户需要在 uci 的网站上下载 Windows 专用的 Package，目前已知 Numpy 和 Scipy 不能直接用 PIP 安装。

Unofficial Windows Binaries for Python Extension Packages

不喜欢打乱本机环境的话 Anacoda3 也可以，只要挂了 Proxifier 就可以使用 conda install 来安装 Tensorflow。

安装 CUDA

亲测 CUDA 9.0 不能正常工作，所以需要安装 CUDA 8.0 来支持 Tensorflow-GPU。

进入 CUDA 的下载地址，这个网页会默认引导下载最新版本的 CUDA，根据 nvidia 官方的回答：

https://developer.nvidia.com/cuda-downloads

找到 CUDA 8.0 版本下载链接，根据提示选择系统和安装包格式：

https://developer.nvidia.com/cuda-80-ga2-download-archive

下载完成后直接根据提示安装，记得勾选 ADD TO PATH，这样就不用手动设置系统变量。

安装 CUDNN

CUDNN 必须要是注册会员且填写过调查问卷才能下载使用，本账号已完成过这一流程，直接进入下载地址，找到 CUDA 8.0 对应的版本:

https://developer.nvidia.com/rdp/cudnn-download

下载解压之后是三个文件夹：bin、include、lib，将其 解压到 CUDA 所在的文件夹，与原有的这三个文件夹 合并。

至此就安装完毕了。
